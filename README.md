# 《Tom and Jerry》小游戏
## 说明
- 基于mooc上电子科技大学《程序设计基础（C&C++）》课程的戴波老师在网课中示范讲解的程序改写完善而成。
- 利用适合C初学者的ACLLib库进行简单的图形界面的编写。[ACLLib说明及下载地址](https://github.com/wengkai/ACLLib)。
- 开发平台：Win 10
- 开发环境：Visual Studio 2019
